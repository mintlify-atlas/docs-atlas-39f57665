## What is WeakAuras?

WeakAuras is a powerful and flexible framework that allows the display of highly customizable graphics on World of Warcraft's user interface to indicate buffs, debuffs, and other relevant information. Originally created as a lightweight replacement for Power Auras, it has since evolved to offer extensive functionality while remaining efficient and easy to use.

<Note>
  **Version**: 5.21.2 | **Interface**: 30300 (WotLK 3.3.5a) | **Author**: The WeakAuras Team
</Note>

## Key Benefits for WotLK Players

### Enhanced Awareness

Never miss important procs, cooldowns, or debuffs again. WeakAuras puts critical information front and center, helping you react faster and play better.

### Raid & Dungeon Ready

Integration with boss mods (DBM, BigWigs) allows you to create custom alerts for raid mechanics, helping your team succeed in challenging content.

### Class-Specific Optimization

Track class resources like runes, totems, holy power, and soul shards with specialized displays designed for WotLK's unique class mechanics.

### Lightweight & Stable

Despite its power, WeakAuras is designed to be lightweight with smart loading conditions that only activate displays when needed.

## Quick Access

To open the WeakAuras options window:

<CodeGroup>
```bash Slash Command
/wa
```

```bash Alternative
/weakauras
```
</CodeGroup>

Or click the **minimap icon** for quick access.
